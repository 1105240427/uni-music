# CloudMusic

#### Description

* 网易云音乐页面太好看了,我们来clone一个吧! 有兴趣的小伙伴可以一起加入


#### Technology Stack

* VueJs
* uni-app
* Weex

#### Run

1. git clone 到本地
2. 将项目拖入HbuilderX中运行到手机或模拟器


#### Contribution

1. Fork the repository
2. Create Feat_xxx branch
3. Commit your code
4. Create Pull Request
